### 4.1 `GlobalGameStateContext` 状态管理

绝对不允许玩家跨越进度硬解谜题。通过以下全局变量切断违规路径：

  currentPhase: 1 | 2 | 3 | 4;

  // 已阅读的关键伏笔 ID 列表（控制 InvestigateNode 激活）
  readHooks: string[];
