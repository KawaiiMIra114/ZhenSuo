## 零、概览

Meridian Life Sciences 是安宁深眠诊所的幕后投资方。
在游戏里作为独立域名存在，不显示在桌面、开始菜单、或任何导航里。
唯一发现路径：官网首页底部合作伙伴区块 → 玩家自己在浏览器地址栏输入域名。

域名：`meridian-ls.com`
在Browser组件里注册为可访问域名，渲染 `<Meridian />` 组件。

## 二、Meridian页面整体风格

背景色：`#f8f7f4`（米白，高端咨询公司质感，不是诊所的冷白）
主色：`#1a1a2e`（深蓝黑）
强调色：`#c9a84c`（暗金）
字体：衬线字体为主，数据部分用等宽

整体感觉：一家真实存在的、有钱的、完全合法的生命科学投资基金。
没有任何东西看起来可疑，直到玩家仔细读投资组合。

## 四、触发逻辑

**发现Meridian：**
玩家在Browser地址栏输入 `meridian-ls.com` 或 `www.meridian-ls.com` → 渲染Meridian组件。
同时触发：`addFact('meridian_visited')`（新Fact，替换原有的`meridian_suspicious`）

**结局后状态变化：**
Meridian组件读取 `hasFact('erasureActive')` 或 `GameContext` 里的 `erasureActive` 状态。
若为true，安宁诊所卡片切换到TERMINATED状态（见3.4）。

**无其他触发，无碎片，无密码。**
这是纯叙事内容，不影响任何游戏机制。

## 五、Browser组件更新

在 `Browser.tsx` 的URL解析逻辑里新增：

```typescript
if (host.includes('meridian-ls.com')) {
    target = 'meridian';
}
```

`browserApp` 状态类型扩展：
```typescript
'clinic' | 'forum' | 'oa' | 'meridian'
```

Desktop.tsx 里新增对应的渲染分支：
```tsx
{browserApp === 'meridian' && (
    <Browser title="MERIDIAN LIFE SCIENCES" defaultUrl="https://www.meridian-ls.com">
        <Meridian />
    </Browser>
)}
```

Meridian**不需要**`forum_url_discovered`类型的Fact门控，任何时候输入域名都可以访问。

## 六、注意事项

1. 安宁诊所卡片的TERMINATED状态只在`erasureActive`为true时显示，游戏进行中始终显示Active。
2. PROJECT DELTA卡片的遮罩文字实际上存在于DOM里，只是颜色和背景相同导致不可见。这是有意设计——如果有玩家真的去检查页面源码（现实浏览器里），他们会看到遮罩下面的内容。这条彩蛋不需要在游戏内实现，只是一个设计意图备注。
3. 页脚的`We remember everyone.`和致谢页面的相同文字形成呼应，是有意为之。
